## 1. IPアドレスとは？ - ネットワーク上の「住所」

**IPアドレス (Internet Protocol Address)** とは、ネットワークに接続された機器（PC、スマホ、Raspberry Piなど）に割り当てられる、**ユニークな「住所」**のようなものです。

- **例**: `192.168.1.10`

家（ネットワーク）の中にあるたくさんの部屋（機器）の中から、目的の部屋（Raspberry Pi）を見つけるために、この「住所」が必要になります。

SSH接続するには、まずRaspberry PiのIPアドレスを知る必要があります。

---

## 2. Raspberry PiのIPアドレスを見つける方法

Raspberry Piが自宅のWi-Fiに接続されると、ルーター（Wi-Fiの親機）からIPアドレスが自動的に割り当てられます。これを見つける方法はいくつかあります。

### 方法1: ルーターの管理画面で確認する（推奨）

一番確実な方法です。

1.  **PCのブラウザを開きます。**
2.  アドレスバーに、お使いのルーターのIPアドレスを入力します。一般的には以下のいずれかです。
    - `192.168.1.1`
    - `192.168.0.1`
    - `192.168.11.1`
    - （ルーターの取扱説明書や本体側面にも記載されています）

3.  ログイン画面が表示されたら、ユーザー名とパスワードを入力してログインします。
    - （これも取扱説明書や本体に記載されています）

4.  管理画面の中から、「DHCPクライアント」「接続デバイス一覧」「LAN側デバイス」といったメニューを探します。

5.  デバイス一覧の中に、ホスト名が `raspberrypi` となっている機器があります。そこに表示されているIPアドレスが、あなたのRaspberry Piのものです。

    | ホスト名 | IPアドレス |
    | :--- | :--- |
    | `raspberrypi` | `192.168.1.15` ← これ！ |
    | `DESKTOP-ABCDE` | `192.168.1.12` |
    | `iPhone` | `192.168.1.18` |

### 方法2: `arp`コマンドを使う（中級者向け）

PCのコマンドライン（ターミナル）から探す方法です。

1.  **PCでコマンドプロンプト（Windows）またはターミナル（Mac）を開きます。**
2.  まず、自分のPCのIPアドレスを確認します。
    - **Windows**: `ipconfig`
    - **Mac**: `ifconfig | grep "inet "`
3.  ネットワーク内の全デバイスに通信を試みます。
    - **Windows**: `for /L %i in (1,1,254) do @ping -n 1 192.168.1.%i -w 100 > nul` （`192.168.1`の部分は自分の環境に合わせる）
    - **Mac**: `for i in {1..254}; do ping -c 1 192.168.1.$i & done`
4.  最後に、`arp -a` コマンドを実行すると、ネットワーク内のデバイスのIPアドレスとMACアドレス（物理的な住所）の一覧が表示されます。Raspberry PiのMACアドレスは通常 `b8:27:eb` や `dc:a6:32` で始まることが多いので、それをヒントに探します。

### 方法3: ネットワークスキャナーアプリを使う

スマホアプリの「Fing」などを使うと、簡単にネットワーク内のデバイスを見つけられます。

**💡 ポイント**
見つけたIPアドレスは、ルーターの設定で**固定（静的IP）**にしておくと、毎回変わらなくなるので便利です。ルーターの管理画面で「IPアドレス固定」「DHCP固定割り当て」などの設定を探してみてください。

## 3. SSHで接続しよう！ - ターミナルを開く

IPアドレスがわかったら、いよいよSSH接続です。お使いのPCのOSによって使うツールが異なります。

### Windowsの場合

Windows 10/11には、標準で **OpenSSHクライアント** が搭載されています。

1.  **「Windowsキー + R」** を押し、「ファイル名を指定して実行」ウィンドウを開きます。
2.  `cmd` と入力してEnterキーを押し、**コマンドプロンプト**を起動します。
3.  以下のコマンドを入力してEnterキーを押します。

    ```bash
    ssh pi@192.168.1.15
    ```

    - `pi` の部分は、OSインストール時に設定した**ユーザー名**です。
    - `192.168.1.15` の部分は、先ほど見つけた**Raspberry PiのIPアドレス**です。

4.  初回接続時には、以下のような確認メッセージが表示されます。

    ```
    The authenticity of host '192.168.1.15 (192.168.1.15)' can't be established.
    Are you sure you want to continue connecting (yes/no/[fingerprint])?
    ```

    これは「初めて接続する相手だけど、本当に信頼していい？」という確認です。`yes` と入力してEnterキーを押します。

5.  次に、パスワードの入力を求められます。

    ```
    pi@192.168.1.15's password:
    ```

    OSインストール時に設定した**パスワード**を入力します。**画面には何も表示されませんが、入力はされています。** 入力し終わったらEnterキーを押します。

6.  以下のような表示に変われば、**SSH接続成功**です！

    ```
    pi@raspberrypi:~ $
    ```

    これで、あなたのPCはRaspberry Piの中に入り込みました。ここから先に入力するコマンドは、すべてRaspberry Pi上で実行されます。

### Macの場合

macOSには標準で **ターミナル** があります。

1.  `Command + Space` を押し、Spotlight検索で `ターミナル` と入力して起動します。
2.  Windowsの場合と同じコマンドを入力します。

    ```bash
    ssh pi@192.168.1.15
    ```

3.  初回接続の確認、パスワード入力も同様です。

4.  プロンプトが `pi@raspberrypi:~ $` に変われば成功です。

---

## 4. 最初のコマンド - システムを最新にしよう

SSH接続できたら、まずはお決まりのコマンドを実行して、システムを最新の状態に保ちましょう。

以下のコマンドを1行ずつコピーして、ターミナルに貼り付け、Enterキーを押してください。

```bash
# パッケージリストを更新
sudo apt update

# パッケージをアップグレード
sudo apt full-upgrade -y
```

- `sudo` は「スーパーユーザー（管理者）として実行します」という意味のおまじないです。
- `apt` は、Raspberry Piのソフトウェアを管理するシステムです。

この処理には数分〜数十分かかることがあります。完了するまで待ちましょう。
