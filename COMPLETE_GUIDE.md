## 1. システム概要
「BCNOFNe system（通称: shipOS）」は、Raspberry Pi 4Bを中核とした、完全自律型のAIオペレーティングシステムです。
単なる自動化ツールではなく、状況を自己判断し、必要に応じて自身のコードさえも書き換える「自己進化能力」を備えたパートナーです。

### 主要コンポーネント
- **Core AI (BCNOFNe)**: GPT-4oを脳とし、検索、ファイル操作、コード実行を行う自律エージェント。
- **shipOS Interface**: 船（Ship）をテーマにしたUI（OLED）、音声、LINE連携。
- **Advanced Analytics**: 🚗 OBD走行データ分析（5MT特化）および 🔥 焼却炉計器OCR。

## 2. LINE コマンドリファレンス
LINEはAIへの「航海指示」と「状態確認」のメインインターフェースです。

### ⚓ 基本コマンド
| コマンド | 内容 |
| :--- | :--- |
| `状態` / `status` | システムの現在の稼働状況を確認します。 |
| `停止` / `stop` | AIの自律ループを一時停止します。 |
| `再開` / `start` | AIの自律ループを開始・再開します。 |
| `version` | 現在のシステムバージョン情報を取得します。 |

### 🧭 shipOS モード管理
`モード [モード名]` または `mode [mode_name]` と送信することで、AIの動作プロファイルを切り替えます。
- `autonomous` (自律航海): 通常の自律動作モード。
- `user_first` (入港待機): ユーザーとの対話を最優先するモード。
- `maintenance` (ドック入り): 自己改善やシステム更新を行うモード。
- `power_save` (停泊): 消費電力を抑える低活動モード。

### 🏥 システム状態・ログ
| コマンド | 内容 |
| :--- | :--- |
| `健康` / `health` | ハードウェア（熱、ディスク、サービス）のヘルスチェック結果を表示。 |
| `航海日誌` / `logbook` | 本日のAIの活動サマリーを表示します。 |
| `今日なにした？` | AIが今日実行したタスクをカジュアルに報告します。 |

### 🛠️ 運用コマンド
- **指示の送信**: テキストをそのまま送ると、AIはそれを「新しい目標」として認識し、解決策を自律的に考え実行します。
- **課金承認**: 「💰 課金確認」が届いたら、下部のメニューから `✅ 許可` または `❌ 拒否` を選択してください。

## 3. 高度な解析機能

### 🚗 OBD 走行分析 (5MT 高度化)
マニュアル車に特化した走行評価機能です。
- **使い方**: 走行データの `.csv` ファイルをLINEに送信してください。
- **判定**: `S` (神運転), `A`, `B`, `C` でランク付けされます。
- **神運転ゲート**: Sランク獲得には、スコアだけでなく「急加減速の少なさ」などの厳しい実測条件をクリアする必要があります。
- **環境補正**: 緯度経度からカーブや勾配を検知し、「安全のための減速」を自動で加点補正します。

### 🔥 焼却炉 OCR 解析
焼却炉の計器をスマートにデジタル化します。
- **使い方**: 計器盤の「写真」をLINEに送信してください。
- **処理**: OpenCVとTesseractを使用して数値を抽出し、`/mnt/hdd/incinerator` に保存します。

## 4. 自己進化と安全機能

### 🧠 自己改善 (Self-Improvement)
「自分のコードを直して」と指示すると、AIは自身のソースコードを分析し、バグ修正や性能向上を自動で適用します。
- **安全策**: 修正前に必ずバックアップを作成し、リスクが高い場合は実行せずに提案のみを行います。

### 🛡️ コストガード
OpenAI APIの使いすぎを防ぐ多層防御です。
- **自動停止**: 1日のコスト上限（通常日 300円 / 特別日 1000円）に達すると、AIは全活動を停止します。
- **リアルタイム通知**: 閾値を超えた時点でLINE/Discordにアラートを送信します。

## 5. ハードウェア・メンテナンス

### 📟 OLEDディスプレイ
Raspberry Pi本体の小型ディスプレイに、現在のモード、CPU温度、IPアドレス、AIの機嫌などがリアルタイム表示されます。

### 💾 ストレージ管理 (Hybrid Storage)
- **SSD**: 高速なシステム・プログラム用。
- **HDD (NAS)**: ログや記憶データの大容量アーカイブ用。
- **NASアクセス**: 同じネットワークのPCから `\\<IPアドレス>\nas` でアクセス可能です。
