# Raspberry Pi 完全セットアップガイド - BCNOFNeシステムへの第一歩

---

## 🚀 はじめに

このガイドは、**BCNOFNe（ボクノフネ）システム**を動かすための母艦となる **Raspberry Pi（ラズベリーパイ）** の準備を、ゼロから完璧にサポートするためのものです。

パソコンに詳しくない方でも、この手順通りに進めれば、AIサーバーの土台を確実に構築できます。専門用語は都度解説するので、安心して進めてください。

### このガイドで達成できること

1. ✅ Raspberry PiのOSをインストールできる
2. ✅ Raspberry Piの初期設定を完了できる
3. ✅ **SSH**という遠隔操作方法を有効にできる
4. ✅ 自宅のネットワークにRaspberry Piを接続できる

---

## 1. 準備するものリスト

まずは、必要なハードウェアを揃えましょう。

| 必須アイテム | 説明 | 備考 |
| :--- | :--- | :--- |
| **Raspberry Pi 4 Model B** | AIシステムの頭脳。**メモリ4GB以上**を強く推奨します。 | 8GB版が理想的です。 |
| **microSDカード** | OSをインストールする記憶媒体。**32GB以上**、**Class 10 / U1**以上を推奨。 | 高速なものほど快適です。 |
| **電源アダプター** | Raspberry Pi 4B専用の **USB Type-C、5V/3.0A** のもの。 | 電圧不足は動作不安定の原因になります。 |
| **PC（Windows, Mac, Linux）** | microSDカードにOSを書き込むために使います。 | microSDカードリーダーも必要です。 |
| **Mini Tower Kit** | （推奨）CPUを冷却し、状態を表示するOLEDが付いたケース。 | 今回のシステムに最適化されています。 |

| あると便利なアイテム | 説明 |
| :--- | :--- |
| **モニター** | 初回設定時に画面を確認するために使います。 | microHDMIケーブルが必要です。 |
| **USBキーボード・マウス** | 初回設定時に操作するために使います。 |

**💡 ポイント**
初回設定さえ終われば、モニターやキーボードは不要になります。その後は、お使いのPCからすべての操作が可能です。

## 2. OSのインストール - Raspberry Pi Imagerを使おう

Raspberry Piの頭脳となるOS（基本ソフト）をmicroSDカードにインストールします。

### ステップ1: Raspberry Pi Imagerをダウンロード

1.  お使いのPCで、[Raspberry Pi公式サイト](https://www.raspberrypi.com/software/)にアクセスします。
2.  お使いのOS（Windows, macOS, Ubuntu）に合った「Raspberry Pi Imager」をダウンロードし、インストールします。

### ステップ2: OSをmicroSDカードに書き込む

1.  PCにmicroSDカードを接続します。
2.  Raspberry Pi Imagerを起動します。

3.  **「OSを選ぶ」** をクリックし、以下の順で選択します。
    - `Raspberry Pi OS (other)` → `Raspberry Pi OS (Legacy, 64-bit) Lite`
    - **Lite版**は画面表示がシンプルな分、AIサーバーとしての動作が軽快になります。

4.  **「ストレージを選ぶ」** をクリックし、接続したmicroSDカードを選択します。

5.  **歯車アイコン（詳細設定）** をクリックします。**ここが最重要です！**

    - **ホスト名を設定する**: `raspberrypi.local` のままでOKです。
    - **SSHを有効化する**: ✅ **必ずチェックを入れてください。**
        - `パスワード認証を使う` を選択します。
    - **ユーザー名とパスワードを設定する**: 
        - **ユーザー名**: `pi` （または好きな名前）
        - **パスワード**: **絶対に忘れない、かつ推測されにくいもの**を設定してください。
    - **Wi-Fiを設定する**: ✅ **チェックを入れます。**
        - **SSID**: あなたの家のWi-Fiネットワーク名
        - **パスワード**: Wi-Fiのパスワード
        - **Wi-Fiを使う国**: `JP` (Japan)
    - **ロケール設定をする**: 
        - **タイムゾーン**: `Asia/Tokyo`
        - **キーボードレイアウト**: `jp`

6.  設定が終わったら **「保存」** をクリックします。

7.  **「書き込む」** をクリックします。確認メッセージが出たら「はい」を選択します。
    - microSDカードの中身はすべて消去されます。

8.  書き込みが完了するまで数分待ちます。「書き込みが完了しました」と表示されたら成功です！

---

## 3. Raspberry Piの起動と初回設定

1.  PCからmicroSDカードを取り出し、Raspberry Pi本体のカードスロットに挿入します。
2.  （推奨）Mini Tower KitなどのケースにRaspberry Piを組み込みます。
3.  電源アダプターを接続すると、自動的にRaspberry Piが起動します。
    - 基板上のLEDが点滅し始めます。

4.  **初回起動時は、5分ほど待ちましょう。**
    - この間に、システムが自動で初期設定を行い、Wi-Fiに接続します。

## 4. SSHとは？ - 未来の扉を開く鍵

**SSH (Secure Shell)** とは、一言でいうと「**ネットワーク経由で他のコンピュータを安全に遠隔操作するための仕組み**」です。

![SSHのイメージ](https://i.imgur.com/xG4f4hS.png)

これを使うことで、あなたは自分のPCから、Raspberry Piの中に入り込んでコマンドを実行できます。まるで魔法のようです。

### なぜSSHが必要なの？

- **モニターやキーボードが不要になる**: 一度設定すれば、Raspberry Piは家のどこかに置いておくだけでOK。
- **コマンドのコピー＆ペーストが楽**: PCでコピーしたコマンドを、そのままRaspberry Pi上で実行できます。
- **効率的な作業**: これからのインストール作業は、すべてSSH経由で行います。

OSのインストール時にSSHを有効にしたので、あなたのRaspberry Piはすでに遠隔操作を受け付ける準備ができています。
